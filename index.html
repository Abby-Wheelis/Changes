What are the pro's and cons of 3 visualizations techniques discussed in chapter 8?

  1. Time Series Line Charts:
    • This technique plots equally spaced time intervals on the x axis and the magnitude of the variable that you wish to represent on the y axis.
    • These charts allow for the general trends in the data to be seen.
    • However, it makes it difficult for the extra noise to be avoided and sometimes allows the viewer to miss kew cyclical patterns that help explain the data.
    • Adding a moving average curve or index origin, amoung other strategies can help to make this chart more useful and help to show trends.
  2. Seasonal Subseries Chart
    • These charts compare values at the same points in the cycle for cyclical data.
    • This is useful to compare cyclical relationships, so that the values for each of these cycles can be compared, such as employment in July every year.
    • This makes it easy to see if data is cyclical or not.
    • However, it is not possible to see all of the trends in the data because it is broken up.
  3. Logarithmic Scale
    • A logarithmic scale can be applied to the variable that is represented on the y axis to make an exponential data set linear.
    • This allows you to determine if a data set is exponential.
    • However, it may downplay other variations in the data which may need to be examined.
